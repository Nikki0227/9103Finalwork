## Part 1 – Imaging Technique Inspiration

### Inspiration Source

#### p5.js Data Structure Garden
Our main inspiration came from the case study “Data Structure Garden” on the P5.js website. This work uses smooth animations, soft colors, and rich interactive effects arranged in layers to create the atmosphere of a dream garden. 

I was particularly struck by how simple interactions could elicit beautiful and dynamic flower-like responses, thus providing a sense of care for the users. Our design team’s goal was to build an interactive musical garden that showcases the changing seasons, operated via mouse and keyboard.


#### The aspect of the example i want to incorporate 
I want to incorporate the interactive plant response system from the "Garden Data Structures p5.js" document into our project. In our musical garden art work, user inputs, such as mouse movements and clicks, will trigger flowering and ambient sounds.

#### Why I believe it's a beneficial technique
This technique is useful because it creates a stronger connection between the user and the digital environment, making the garden responsive, immersive, and alive, while supporting the work’s interest in interactive tools and user participation.

## Part 2 – Coding Technique Exploration

### Coding Technique

####  Mouse, Keyboard, and Text Interaction in p5.js
This work will use mouse input functions such as `mousePressed()`, `keyPressed()`, and dynamic captions to create responsive audiovisual interaction in p5.js.

#### Reference Link
[mousePressed() Example](https://p5js.org/reference/p5/mousePressed/)
[keyPressed() Example](https://p5js.org/reference/p5/keyPressed/)


### How this technique helps my project
This programming technique combines user interaction with visual and auditory feedback. Mouse clicks make flowers bloom and trigger sound effects, while keyboard input controls seasonal changes or environmental conditions. Dynamic captions can display prompts or interactive hints to help users better understand and use the interactive garden system.
